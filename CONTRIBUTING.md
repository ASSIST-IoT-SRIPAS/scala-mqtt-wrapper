## Development environment
For development, run `app.sh` and `env.sh` scripts:
```
# first terminal

./dev/scripts/app.sh
```

```
# second terminal

./dev/scripts/env.sh
```

The `app.sh` script will start the application in the development mode.
The `env.sh` script will start the development environment.
